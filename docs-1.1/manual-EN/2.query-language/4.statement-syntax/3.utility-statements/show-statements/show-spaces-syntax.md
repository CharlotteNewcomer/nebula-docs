# SHOW SPACES Syntax

```ngql
SHOW SPACES
```

`SHOW SPACES` lists the SPACES on the **Nebula Graph** cluster.

For example:

```ngql
nebula> SHOW SPACES;
====================
| Name             |
====================
| basketballplayer |
--------------------
```
